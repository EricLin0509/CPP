# 嵌套循环

嵌套循环是指在循环体内再嵌套一个循环

嵌套循环可以用来处理多维数据，例如二维数组、多维矩阵等

## 语法

```cpp
/* 外层循环 */
for (初始条件1; 终止条件1; 增量1)
{
    /* 内层循环 */
    for (初始条件2; 终止条件2; 增量2)
    {
        // 循环体
    }
}
```

- **注意**: 通常情况下，外层循环与内层循环的**初始条件的变量名一般不同**，以避免混淆

## 示例

我们有个输出一至十的代码

```cpp
for (int i = 1; i <= 10; i++)
{
    cout << i << " ";
}
cout << "\n"
```

现在我们想让它输出三次

我们就可以使用嵌套循环

```cpp
for (int i = 0; i < 3; i++)
{
    for (int j = 1; j <= 10; j++)
    {
        cout << j << " ";
    }
    cout << "\n";
}
```

- 外层循环：控制输出的次数
- 内层循环：控制输出的数字

## 经典问题: 输出等腰三角形

使用嵌套循环，我们可以输出等腰三角形

```
    *
   ***
  *****
 *******
*********
```

这里只要求输出五行

```cpp
#define MAX_ROWS 5
```

### 初始化空格数量

初始的空格数量应为 `n - 1`，其中 `n` 是需要输出的行数

```cpp
int spaces = MAX_ROWS - 1;
```

### 遍历行

我们使用外层循环遍历行

```cpp
for (int i = 1; i <= MAX_ROWS; i++)
{
    
}
```

- **注意**: 外层循环的初始条件为 `1`，为了给输出 `*` 留出空间

### 输出空格

```cpp
for (int i = 1; i <= MAX_ROWS; i++)
{
    for (int j = 0; j < spaces; j++)
    {
        cout << " ";
    }
}
```

### 输出星号

通过观察上面的输出，我们可以发现星号的数量为 [1, 3, 5, 7, 9]

我们可以从中发现规律，星号的数量为 `2 * i - 1`

```cpp
for (int i = 1; i <= MAX_ROWS; i++)
{
    for (int j = 0; j < spaces; j++)
    {
        cout << " ";
    }

    for (int k = 0; k < (2 * i - 1); k++)
    {
        std::cout << "*";
    }
}
```

### 输出换行并减少空格数量

通过观察上面的输出，我们可以发现空格的数量为 [4, 3, 2, 1, 0]

我们可以从中发现规律，空格的数量变化为 `spaces - 1`

```cpp
for (int i = 1; i <= MAX_ROWS; i++)
{
    for (int j = 0; j < spaces; j++)
    {
        cout << " ";
    }

    for (int k = 0; k < (2 * i - 1); k++)
    {
        std::cout << "*";
    }

    spaces--;
    std::cout << "\n";
}
```

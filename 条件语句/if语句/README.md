# if语句

`if`语句用于根据条件执行不同的代码块

## 语法

```cpp
if (条件表达式)
{
    // 如果条件为真，执行这里的代码
}
else if (条件表达式) // 可选
{
    // 如果第一个条件为假，第二个条件为真，执行这里的代码
}
else // 可选
{
    // 如果条件为假，执行这里的代码
}
```

条件表达式可以是任何可以转换为布尔值的表达式，例如：

- 逻辑表达式：`a > b && c < d`
- 关系表达式：`a == b`
- 布尔表达式：`true` 或 `false`
- 逻辑运算符：`!a` 或 `a && b` 或 `a || b`

## 示例

假设我们现在要判断用户输入的年龄来判断是否成年

### 获取用户输入

```cpp
/* 不推荐直接使用 using namespace std */ 
using std::cin;
using std::cout;

int age = -1;
cout << "请输入你的年龄: ";
cin >> age;
```

### 判断是否成年

使用`if`语句来判断是否成年

```cpp
if (age >= 18)
{
    cout << "你已经成年了！\n";
}
else if (age < 0)
{
    cout << "你输入的年龄不合法！\n";
}
else
{
    cout << "你还没有成年！\n";
}
```

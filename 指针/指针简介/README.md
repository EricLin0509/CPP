# 指针简介

指针是C++语言中非常重要的概念，它允许我们直接访问内存中的数据

指针的使用可以提高程序的效率，也可以使程序更加灵活

## 语法

```cpp
类型 *变量名;
```

- `*` 表示指针

## 示例

假设现在有个变量`int a = 10;`，我们想要这个变量的地址

```cpp
int a = 10;
int *p = &a;
```

将它转换为汇编语言

```asm
; int a = 10
SUB RSP, 4 ; 将 RSP 寄存器减4，为变量 a 分配内存
MOV DWORD PTR [RSP], 10 ; 将 10 存入 RSP 寄存器指向的内存地址

; int *p = &a
LEA RAX, [RSP] ; 将 RSP 寄存器的值赋给 RAX 寄存器
```

### 指针的数据大小

我们可以使用 `sizeof` 操作符来获取变量的大小

```cpp
std::cout << "The size of pointer is: " << sizeof(p) << "\n";
```

在32位系统上，这个值应为4字节

```
The size of pointer is: 4
```

在64位系统上，这个值应为8字节

```
The size of pointer is: 8
```

### 解引用

如果我们想访问某个指针指向的内存地址里面的值，可以使用解引用操作符`*`

```cpp
int a = 10;
int *p = &a;
int b = *p; // b = 10
```

那么 `int b = *p;` 相当于

```asm
MOV RBX, [RAX] ; 将 `RAX` 的数值指向的内存地址里面的数值存进RBX寄存器
```

#### MOV 与 LEA 的区别

- `MOV` 支持从寄存器到寄存器，寄存器到内存，内存到寄存器，寄存器到寄存器等操作
- 而 `LEA` 只计算地址，不访问内存

# 数组排序

现在有一个未排序的数组

```cpp
int arr[] = {10, 1, 5, 7, 8, 2, 4, 6, 3, 9};
```

要求对这个数组进行排序

这里使用冒泡排序算法

## 1.外层循环

控制排序轮数 (共 `n-1` 轮，`n` 为数组长度)

```cpp
int n = sizeof(arr) / sizeof(arr[0]); // 这里 n = 10

for (int i = 0; i < n - 1; i++)
{

}
```

- `i < n - 1`：由于最后一个元素不需要比较，所以只需要遍历 `n-1` 次

## 2.内层循环

每轮遍历未排序部分 (每轮后右侧元素已排序)

```cpp
for (int i = 0; i < n - 1; i++)
{
    for (int j = 0; j < n - 1 - i; j++)
    {

    }
}
```

## 3.比较相邻元素

每次比较相邻的元素，若顺序错误则交换

```cpp
for (int i = 0; i < n - 1; i++)
{
    for (int j = 0; j < n - 1 - i; j++)
    {
        if (arr[j] > arr[j + 1])
        {

        }
    }
}
```

## 交换元素

这里可以使用异或运算符进行交换，这样可以不使用任何临时变量

```cpp
void quick_swap(int *a, int *b)
{
    if (a == NULL || b == NULL) return; // 防止空指针异常
    if (a == b) return; // 防止交换同一个元素

    *a ^= *b;
    *b ^= *a;
    *a ^= *b;
}
```

```cpp
for (int i = 0; i < n - 1; i++)
{
    for (int j = 0; j < n - 1 - i; j++)
    {
        if (arr[j] > arr[j + 1])
        {
            quick_swap(arr + j, arr + j + 1); // 因为数组名即指针，我们可以直接传递计算好的地址
        }
    }
}
```

### 原理

对于两个不同的数 `a` 和 `b`，存在公式 `a ^ b ^ b = a`

这是因为 `b ^ b = 0`、`a ^ 0 = a` 且异或运算满足交换律

那么 `quick_swap` 的步骤为

1. `*a ^= *b`：`*a` 变为 `*a ^ *b`
2. `*b ^= *a`：`*b` 变为 `*b ^ (*a ^ *b) = *a ^ *b ^ *b = *a`
3. `*a ^= *b`：`*a` 变为 `(*a ^ *b) ^ *a = *b ^ *a ^ *a = *b`

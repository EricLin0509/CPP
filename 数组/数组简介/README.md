# 数组简介

数组是C++中一种基本的数据结构，用于存储相同类型的元素。数组的元素可以通过索引访问，索引是从0开始的。

## 语法

```cpp
数据类型 数组名[数组大小];

数据类型 数组名[] = {元素1, 元素2, 元素3, ...};
```

- 如果显式初始化数组，数组大小可以省略，编译器会根据初始化的元素个数自动计算数组大小

```cpp
数据类型 数组名[] = {元素1, 元素2, 元素3, ...};
```

- 如果隐式初始化数组，数组大小必须显式指定

```cpp
数据类型 数组名[数组大小];

数组名[索引] = 元素;
```

## 示例

### 初始化数组

```cpp
int arr[3] = {1, 2, 3};
```

### 访问数组元素

使用以下代码访问数组

```cpp
std::cout << arr << std::endl;
```

此时的输出为

```
0x7fffffffd64c
```

这里的输出是数组的地址，而不是数组的元素。这是因为数组名实际上是一个指向数组首元素的指针

因此要访问数组的元素，需要使用数组名加上索引

```cpp
std::cout << arr[0] << std::endl;
std::cout << arr[1] << std::endl;
std::cout << arr[2] << std::endl;
```

此时的输出为

```
1
2
3
```

- **注意**: 数组的索引**从0开始**

### 修改数组元素

使用数组名+索引修改数组元素

```cpp
arr[0] = 10;
```

此时数组变为

```
10 2 3
```

# 算术运算符

## 加法运算符

使用 `+` 运算符可以对两个整数进行加法运算

```cpp
int a = 5;
a = a + 3; // a = 8
```

### 自增运算符

如果只需要对一个变量进行加一操作，可以使用 `++` 运算符

```cpp
int a = 5;
a++; // a = 6
++a; // a = 7
```

区别
- `a++` 先使用 `a` 的值，再对 `a` 进行加一操作
- `++a` 先对 `a` 进行加一操作，再使用 `a` 的值

## 减法运算符

使用 `-` 运算符可以对两个整数进行减法运算

```cpp
int a = 5;
a = a - 3; // a = 2
```

### 自减运算符

如果只需要对一个变量进行减一操作，可以使用 `--` 运算符

```cpp
int a = 5;
a--; // a = 4
--a; // a = 3
```

区别
- `a--` 先使用 `a` 的值，再对 `a` 进行减一操作
- `--a` 先对 `a` 进行减一操作，再使用 `a` 的值

## 乘法运算符

使用 `*` 运算符可以对两个整数进行乘法运算

```cpp
int a = 5;
a = a * 3; // a = 15
```

## 除法运算符

使用 `/` 运算符可以对两个整数进行除法运算

```cpp
int a = 5;
a = a / 3; // a = 1
```

## 求余运算符

使用 `%` 运算符可以对两个整数进行求余运算

```cpp
int a = 5;
a = a % 3; // a = 2
```

## 简化写法

如果需要对一个变量进行指定数值的运算并赋值，可以使用以下简写形式

```cpp
<变量名> <运算符>= <操作数>
```

```cpp
a += 3;
a -= 3;
a *= 3;
a /= 3;
a %= 3;
```

## 指正

在使用自增或自减运算符，如果出现类似以下公式 (**FUCK 谭浩强**)

```cpp
i++ + ++i;

i = i++ + 1;
```

这属于**未定义行为 (Undefined Behavior)**

因为它多次修改同一个标量变量，且这些修改操作之间不存在控制性的顺序点或顺序关系

由于编译器可以自由地以任何方式优化或重排底层机器指令，这段代码可能产生完全不可预测的结果，在不同编译器上得出不同值，或者导致程序不稳定

详情请参考 [Stack Overflow](https://stackoverflow.com/questions/10778627/why-is-i-i-1-undefined-behavior-in-c11)

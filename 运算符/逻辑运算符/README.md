# 逻辑运算符

逻辑运算符用于比较两个表达式，返回一个布尔值 (`true` 或 `false`)

常见的逻辑运算符包括：

- `&&`：逻辑与运算符，两个表达式都为 `true` 时返回 `true`，否则返回 `false`
- `||`：逻辑或运算符，两个表达式中至少有一个为 `true` 时返回 `true`，否则返回 `false`
- `!`：逻辑非运算符，将表达式的布尔值取反，`true` 变为 `false`，`false` 变为 `true`

## 逻辑与运算符

逻辑与运算符 `&&` 用于比较两个表达式，两个表达式都为 `true` 时返回 `true`，否则返回 `false`

假设我们要判断某个数字是否在某个范围内，可以使用逻辑与运算符来实现：

```cpp
int num = 10;

if (num >= 0 && num <= 100)
{
    // num 在 0 到 100 之内
    std::cout << "num 在 0 到 100 之内" << std::endl;
}
```

## 逻辑或运算符

逻辑或运算符 `||` 用于比较两个表达式，两个表达式中至少有一个为 `true` 时返回 `true`，否则返回 `false`

假设我们要判断某个数字是否不在两个不相交的范围内，可以使用逻辑或运算符来实现：

```cpp
int num = 10;

if (num < 0 || num > 100)
{
    // num 不在 0 到 100 之内
    std::cout << "num 不在 0 到 100 之内" << std::endl;
}
```

## 逻辑非运算符

逻辑非运算符 `!` 用于将表达式的布尔值取反，`true` 变为 `false`，`false` 变为 `true`

假设我们要判断某个数字不在某个范围内，可以使用逻辑非运算符来实现：

```cpp
int num = 10;

if (!(num < 0 || num > 100))
{
    // num 不在 0 到 100 之内
    std::cout << "num 在 0 到 100 之内" << std::endl;
}
```

# 三元运算符

三元运算符 (Ternary Operator) 是一种简洁的条件表达式，用于在一行代码中执行条件判断

## 语法

```cpp
条件表达式 ? 表达式1 : 表达式2
```

## 示例

假设现在我们要判断用户输入是否为偶数，如果是偶数则输出“偶数”，否则输出“奇数”

```cpp
int num = 0;
std::cout << "请输入一个数字: ";
std::cin >> num;

if (num & 1 == 0)
{
    std::cout << "偶数" << std::endl;
}
else
{
    std::cout << "奇数" << std::endl;
}
```

我们可以使用三元运算符来简化这个条件判断

```cpp
int num = 0;
std::cout << "请输入一个数字: ";
std::cin >> num;

std::cout << ((num & 1) == 0 ? "偶数" : "奇数") << std::endl;
```

- `num & 1` 是一个位运算符，用于检查 `num` 是否为奇数
    - **注意**: 位运算符的优先级比 `==` (逻辑运算符) 低，因此需要使用括号来确保运算顺序

# 自定义函数

自定义函数是C++编程中非常重要的概念，它允许你定义自己的函数，以多次调用，提高代码的可重用性和可读性

## 语法

### 声明函数

```cpp
返回类型 函数名(参数列表);
```

- 返回类型：函数返回的值的数据类型，可以是任何数据类型，或者无返回值 (`void`)
- 函数名：函数的名称，在C++中可以相同 (因为C++支持函数重载)，这种情况下只需要保证相同函数名的函数中的参数列表不同即可
- 参数列表：函数的参数列表，可以是无参数的 (`void`)，也可以有多个参数
- **必须在函数调用之前声明函数**

### 定义函数

定义函数等于声明+实现函数体

```cpp
返回类型 函数名(参数列表)
{
    // 函数体
}
```

## 示例

现在要计算三次加法，代码如下

```cpp
int sum1 = 0;
int sum2 = 0;
int sum3 = 0;

sum1 = 2 + 3;
std::cout << sum1 << "\n";

sum2 = 4 + 5;
std::cout << sum2 << "\n";

sum3 = 6 + 7;
std::cout << sum3 << "\n";
```

这样会导致代码冗余，我们可以定义一个函数来计算加法

### 定义函数

```cpp
void print_sum(int a, int b)
{
    std::cout << a + b << "\n";
}
```

### 调用函数

```cpp
print_sum(2, 3);
print_sum(4, 5);
print_sum(6, 7);
```

- 调用函数时，提供的参数必须与函数声明时的**参数类型和数量一致**

这样代码就变得简洁了

# 常量

在C++中，常量用于表示不会改变的值

常量可以是整数、浮点数、字符、字符串等。常量的声明和使用方式与变量类似

但常量的值在声明后**不能被修改**

## 场景

假设现在我们想计算一个圆的周长

```cpp
double pi = 3.14;
double radius = 5.0;
double circumference = 2 * pi * radius;
```

但这样会导致一个问题，可能在计算过程中，pi的值被修改了，导致计算结果不准确

```cpp
double pi = 3.14;
pi = 16.0; // 修改了pi的值
double radius = 5.0;
double circumference = 2 * pi * radius; // 计算结果不准确
```

为了解决这个问题，我们可以将pi的值声明为常量

## 声明常量

常量的声明方式与变量类似，但需要在声明时使用关键字`const`

```cpp
const double pi = 3.14;
```

此时我们修改pi的值

```cpp
pi = 16.0; // 编译错误，常量的值不能被修改
```

会出现编译错误，因为常量的值在声明后不能被修改

```
error: 向只读变量‘PI’赋值
```

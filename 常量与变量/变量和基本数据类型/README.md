# 变量和基本数据类型

## 变量

变量是存储数据的容器，用于存储和操作数据

### 声明变量

使用以下代码声明一个变量:

```cpp
<数据类型> <变量名>; // 必须以分号结尾
```

```cpp
int a; // 声明一个整型变量a
```

### 给变量赋值

使用以下代码给变量赋值:

```cpp
<变量名> = <值>; // 变量名 = 值
```

```cpp
a = 10; // 给变量a赋值10
```

### 便捷写法

可以在一行代码中声明和赋值:

```cpp
<数据类型> <变量名> = <值>;
```

```cpp
int a = 10; // 声明和赋值一个整型变量a
```

## 基本数据类型

基本数据类型是C++语言中用于存储基本数据的类型

### 整数类型

整数类型包括以下几种:

- `int`: 32位整数 (-2^31 ~ 2^31-1)
- `short`: 16位整数 (-2^15 ~ 2^15-1)
- `long`: 至少32位整数 (64位系统中通常为64位，范围 -2^63 ~ 2^63-1)
- 在绝大多数场景中，`int` 和 `long` 是最常用的整数类型

```cpp
int a = 10; // 声明和赋值一个整型变量a
short b = 20; // 声明和赋值一个短整型变量b
long c = 30; // 声明和赋值一个长整型变量c
```

### 浮点数类型

浮点数类型包括以下几种:

- `float`: 单精度浮点数 (约6-9位有效十进制数字)
- `double`: 双精度浮点数 (约15-17位有效十进制数字)

```cpp
float a = 10.5; // 声明和赋值一个单精度浮点数a
double b = 20.5; // 声明和赋值一个双精度浮点数b
```

### 字符类型

字符类型只有 `char` 类型

```cpp
char a = 'a'; // 声明和赋值一个字符变量a
```

- 注意：保存单字节值（0-255），通常用于 [ASCII](https://zh.wikipedia.org/wiki/ASCII) 字符

### 布尔类型

布尔类型只有两种值: `true` 和 `false`

```cpp
bool a = true; // 声明和赋值一个布尔变量a
```

### 字符串类型

字符串类型是C++语言中用于存储字符串的类型，但它是一个对象

```cpp
std::string a = "Hello World"; // 声明和赋值一个字符串变量a
```
